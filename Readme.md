# User & Admin Login System

## 📌 Overview
The **Login System** is the entry point for the User and Admin Dashboard. It provides secure authentication and role-based access control, ensuring that users and administrators have distinct privileges and tailored experiences.

---

## 🔐 Features

### 👤 User Login
- **Email & Password Authentication** – Secure login with validation.
- **Forgot Password** – Reset credentials via email.
- **Session Management** – Stay logged in with secure cookies/tokens.
- **Redirect to User Dashboard** – Access personal dashboard after login.

### 🛠 Admin Login
- **Admin Authentication** – Restricted access for administrators only.
- **Role-Based Access Control** – Different permissions for Admin vs User.
- **System Monitoring Access** – Redirect to Admin Dashboard with management tools.
- **Enhanced Security** – Stronger password policies and audit logging.

---

- **Frontend:** THYMELEAF(HTML, CSS, JavaScript (/Angular/)
- **Backend:** Java (Spring Boot) / Tomcat /
